Odoo Connector¶

Odoo Connector est un puissant framework de développement de connecteur bi-directionnel entre Odoo (ERP Open Source) et tout autre logiciel ou service.

Ce module Odoo a un cœur modulaire et générique, capable d’être lui-même étendu grâce à des modules additionnels, pour ajouter des nouvelles fonctions ou des personnalisations.

The development of Odoo Connector has been started by Camptocamp and is now maintained by Camptocamp, Akretion, Acsone and several Contributeurs.

It got a major overhaul in 2017 (Odoo 10.0). A Migration Guide to the new Connector API is available.

Subscribe to the project’s mailing list (name: Connectors)

Découvrez comment Contribuer

Fonctionnalités de base¶

  • 100% Open Source (LGPL version 3): la totalité du code source est disponible sur GitHub

  • Pas uniquement conçu pour connecter Odoo à des backend e-commerce, mais adaptable pour connecter Odoo à tout type de service.

  • Robuste pour des volumétries importantes et faciler à surveiller grâce à la Queue de jobs.

  • Un ensemble flexible de blocs de construction : il n’oblige pas à respecter une implémentation particulière mais laisse le choix final au développeur sur la manière d’utiliser les composants.

  • Voir le Survol du code et les exemples de code
